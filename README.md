# Auditoría de Seguridad: Red Doméstica 🛡️

Este repositorio contiene la metodología y el flujo de trabajo para realizar una auditoría de seguridad preventiva en entornos residenciales. El objetivo es identificar vectores de ataque comunes y aplicar técnicas de *hardening*.

## 📋 Fases del Proyecto

1. **Reconocimiento:** Inventario de activos y revisión de hardware.
2. **Escaneo:** Identificación de hosts y servicios (Puertos abiertos).
3. **Análisis de Vulnerabilidades:** Verificación de protocolos (WPS, UPnP) y cifrado.
4. **Mitigación:** Aplicación de parches, cambios de credenciales y segmentación (VLANs/Guest Network).

## 🛠️ Herramientas Sugeridas
* **Nmap:** Para descubrimiento de red y auditoría de puertos.
* **Wireshark:** Análisis de tráfico y detección de protocolos inseguros.
* **Fing:** Escaneo rápido desde dispositivos móviles.
